# 自动化文档处理系统 - 任务计划

## 项目概述
图书管理督导工作自动化汇总系统，实现Word文档的上传、解析、合并、预览编辑和导出。

## 表格结构（10列）
| 列号 | 字段名 | 说明 |
|------|--------|------|
| 0 | 序号 | 需垂直合并 |
| 1 | 值班助理 | 需垂直合并，按姓名排序 |
| 2 | 日期 | 次要排序字段 |
| 3 | 上书量（本） | 需求和 |
| 4 | 纠错量（本） | 需求和 |
| 5 | 整架范围/整架号 | |
| 6 | 工作地点 | |
| 7 | 值班签到 | |
| 8 | 督导检查情况 | 需合并各楼层问题 |
| 9 | 督导检查情况 | 同上 |

## 启动方式

### 方法1：使用启动脚本
双击 `start.bat` 即可同时启动前后端

### 方法2：手动启动
1. 启动后端：
```bash
cd backend
C:\Users\33006\anaconda3\python.exe -m uvicorn main:app --reload --port 8000
```

2. 启动前端：
```bash
cd frontend
npm run dev
```

### 访问地址
- 前端界面: http://localhost:5173
- 后端API: http://localhost:8000

## 技术栈
- Backend: Python 3.x, FastAPI, python-docx, pandas
- Frontend: React (Vite), Tailwind CSS
- 无数据库，使用临时存储
